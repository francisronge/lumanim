The sentence

L: L is false.

Start with TRUE

If L is true, what L says holds. L says that L is false. Therefore L is false.

Start with FALSE

If L is false, L's claim about itself is correct. Therefore L is true.

The exact pressure

Self-reference combines with a truth predicate and negation. Ordinary evaluation then produces:

TRUE → FALSE
FALSE → TRUE

Diagnostic question

When you meet a semantic paradox, ask: What is the statement allowed to say about its own semantic status?
