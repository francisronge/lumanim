Three origins.
One output.

The enormous formula in the post is not the idea. The idea is that a polynomial map can be perfectly reversible around every individual point—and still forget which of three distant starting points produced the same result.

Predict the decisive failure

Which observation would directly prove that a function has no global inverse?

What the Jacobian actually guarantees

For a differentiable map, the Jacobian matrix is its best linear approximation near one input. When its determinant is nonzero, the inverse function theorem gives a local inverse: zoom in far enough around that input and the map is one-to-one there.

Local

Given the output and the neighborhood or “sheet” it came from, you can recover the nearby input.

Global

Given only the output, you may still have several distant candidate inputs on different sheets.

Steve Smale's formulation asks whether polynomial maps are special enough to close this gap: if the derivative is nonsingular at every point of n-dimensional complex space, must the whole map be one-to-one? In 1998 he listed that as Problem 16.

What the post supplies

Levent Alpöge's post gives an explicit polynomial map F: ℂ³ → ℂ³. You do not need to understand how anyone found it. To check whether it is a counterexample, only two exact questions matter.

Check 1: the local promise holds

Exact symbolic differentiation simplifies the Jacobian determinant to −2 everywhere. It never vanishes, so every point has a locally reversible neighborhood.

Check 2: global uniqueness fails

The three distinct inputs

(0, 0, −1/4), (1, −3/2, 13/2), and (−1, 3/2, 13/2)

all evaluate to (−1/4, 0, 0). Therefore F is not injective and cannot have a global inverse.

See the polynomial from the post 
F(x,y,z) = (
  (1+xy)³z + y²(1+xy)(4+3xy),
  y + 3x(1+xy)²z + 3xy²(4+3xy),
  2x − 3x²y − x³z
)

I reran both checks with exact rational arithmetic and symbolic algebra. The preserved verification script prints det JF = -2 and the same output for all three points. Zihan Zhang independently reproduces the certificate and provides another script.

How certain should you be today?

The finite certificate checks out. Unlike a 100-page proof, this claim reduces to symbolic differentiation and three substitutions. That makes independent verification unusually direct.

But the announcement is hours old. There was no peer-reviewed paper when this lesson was produced on 20 July 2026, and specialist analysis is still developing on MathOverflow. The responsible wording today is: the posted map is an exactly verified counterexample; formal publication and historical assessment are still catching up.

Keep

The Jacobian determinant answers a local question: “Can I reverse the map near this input?” The conjecture claimed that polynomial structure upgrades that into a global answer. Alpöge's map satisfies the local test everywhere but gives one output three origins. That single collision is enough to refute the conjecture in dimension three.
