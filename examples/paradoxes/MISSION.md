# Mission: Paradoxes

## Why
Use Lumanim to make the structure of philosophical paradoxes perceptible—not merely describe them—and build the habit of locating the exact commitments that generate each tension.

## Success looks like
- Experience the Ship of Theseus as two competing identity claims.
- Distinguish different paradox structures instead of treating every puzzle as a generic contradiction.
- Make and reconsider predictions using real ManimGL visuals.
- Use every HTML lesson whether or not the live runtime is available.

## Constraints
- Keep each lesson tightly scoped; this is a compact sequence, not a philosophy survey.
- The learner-facing surface is HTML.
- Real 3b1b/manim ManimGL is mandatory.

## Out of scope
- Surveying every proposed solution to each paradox.
- Treating a contested philosophical response as the single correct answer.
